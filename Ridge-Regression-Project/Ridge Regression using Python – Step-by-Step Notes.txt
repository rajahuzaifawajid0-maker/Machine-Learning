# **Step 10: Model Evaluation**

### **Code**

```python
mse
rmse
mae
r2
```

### **Explanation**

### **MSE**

Mean Squared Error

Measures squared prediction error.

Lower value = Better model.

---

### **RMSE**

Root Mean Squared Error

Shows average prediction error.

Lower RMSE = Better prediction.

---

### **MAE**

Mean Absolute Error

Average absolute difference between actual and predicted salary.

Lower MAE = Better model.

---

### **R² Score**

Shows model accuracy.

Range:

* 0 = Poor
* 1 = Perfect

Higher R² = Better performance.
